# Task Manager Web App

A simple **Vanilla JavaScript Task Manager** with dark mode, local storage, and dynamic task management.

## Features
Add tasks with a title and description
 Edit tasks to update title and description
 Mark tasks as **Done/Undone** (status toggling)
 Delete tasks individually or clear all tasks
 Dark Mode toggle with preference stored in **localStorage**
 Load sample tasks from a mock API
 Persistent storage of tasks using **localStorage**

## How to Run

1. Clone or download the project.
2. Open `index.html` in a web browser.
3. Use the input fields to add new tasks.
4. Click **Done** to mark a task as completed.
5. Click **Edit** to update task details.
6. Click **Delete** or **Clear** to remove tasks.
7. Toggle **Dark Mode** to switch themes.
8. Click **Load** to fetch sample tasks.

## Concepts Implemented

 **DOM Manipulation** – creating, editing, and removing HTML elements dynamically
 **Event Handling** – handling click events for buttons
 **Local Storage** – saving tasks and dark mode preferences
 **Conditional Styling** – changing task appearance based on status
 **Async Functions / Callbacks** – simulating fetching data from an API
